    Our projects

    PocketMine-MP related tools

    • FakeClient
      • An Android app for joining PocketMine servers as a bot. A feature is to show the "outdated server" message when joining unless the server installs a plugin that allows a bot.
      • The bot is designed for debugging. It should be able to show debug data visible to a client.

    PocketMine-MP Zekkou Cake plugins

    • StatsCore
      • A central API for other plugins, used for recording statistics of players and providing APIs like the request API.
      • It also has its own functions, such as the player total online time.
      • Dependency for PocketFactions and PocketRealisticMulti.
    • xEcon
    • PocketFactions
    • EssentialsPE
    • PocketRealisticMulti
      • A lot of realistic functions, separated in there modules composed of even more submodules.
      • The physics module contains gravity, sound waves and ropes submodules.
      • The biology module contains realistic health, hunger, potions (by overriding the beetroot stew) and other submodules.
      • The social module contains permanent death (hardcore pro) and possibly other submodules.
      • This plugin has a dependency of StatsCore.
      • This plugin aims on pushing MCPE to its maximum realism.
    • PixelArt-Image
      • The core generates a PixelArt image shown to a client (selected clients only, without creating a world).
      • An add on is to show pictures downloaded from HTTP.
      • Another add-on may be a forced captcha to avoid bots.
